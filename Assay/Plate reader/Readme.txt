This code is a py code aim to process the plate reader data. The input are the numbered well plan and data from plate reader. The output is the result for each condition
The code is in the example of a set of Alamar Blue data(570nm, 600nm), 96 wells. To use it for a different assay, please rename the dataspace for the assay and define the required calculation function
The software contain 3 departments. The first read in name and datas and cluster them to each corresponding sample. The second cluster samples to each corresponding condition. The third process the data of each condition.
